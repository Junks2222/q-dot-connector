# Q Dot Connector

Personal intelligence & pattern-tracking tool for geopolitical events, Q drops, and symbolic signals.

## Features
- Add entries with smart auto-connection to Q drops + other entries
- 18 curated high-signal Q drops
- LocalStorage persistence + JSON export/import
- Modern dark Tailwind UI
- "Ask AI" simulated analysis

## How to use on GitHub Pages (2 minutes)
1. Go to repo Settings → Pages
2. Source: Deploy from a branch → main → Save
3. Wait 30 seconds — site live at https://Junks2222.github.io/q-dot-connector

## Usage
- Click **New Entry** to add observations
- Connections calculated automatically (keywords + date + strategic rules)
- Click any entry for details + Ask AI
- Export/Import for backup

Built for local use. No backend.

**Pro tip:** Press `?` for Connections view. Press `/` to search.
